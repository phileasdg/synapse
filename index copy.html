How to Use

  • Pan & Zoom: Drag the canvas or use your scroll wheel.

  • Add Node: Click any empty area on the canvas.

  • Connect Nodes: Click one node, then another to link them.

  • Edit Node: Double-click a node or use its hover buttons.

  • Set Root: Right-click a node to set as layout root.

  • LaTeX Math: Use `$...$` for inline and `$$...$$` for block equations.
